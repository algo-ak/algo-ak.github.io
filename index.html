03 · Syllabus

Syllabus

Unit-wise topics and lecture hours.

5 units 46 lecture hours Textbook: CLRS, 3rd Edition
Unit 019 hrs

Foundations of Data Structures and Algorithms

  • Review of Basic Data Structures (Arrays, Linked Lists, Stacks, Queues)
  • Complexity Analysis: Time and Space Complexity, Big-O, Ω, and Θ Notation
  • Sorting Algorithms and Divide and Conquer Algorithms (Bubble sort, Insertion sort, Merge Sort, Quick Sort, Binary Search)
  • Recursion and Backtracking
Unit 0210 hrs

Advanced Data Structures

  • Hashing Techniques and Hash Tables
  • Heaps and Priority Queues, Heap sort, Fibonacci Heap sort
  • Binary Search Trees, AVL Trees, and Red-Black Trees
Unit 0310 hrs

Graph Algorithms

  • Representation of Graphs (Adjacency Matrix/List)
  • Breadth-First Search (BFS) and Depth-First Search (DFS)
  • Minimum Spanning Trees (Prim's and Kruskal's Algorithms)
  • Shortest Path Algorithms (Dijkstra, Bellman-Ford, Floyd-Warshall)
Unit 0412 hrs

Greedy Algorithms and Dynamic Programming

  • Greedy Strategy: Principles and Applications
  • Classic Problems: Huffman Encoding, Job sequencing with deadlines
  • Dynamic Programming: Principles and Patterns
  • Classic Problems: Knapsack, Longest Common Subsequence, Matrix Chain Multiplication, Rod-cutting Problem
Unit 055 hrs

Advanced Topics

  • Introduction to Randomized Algorithms
  • NP-Completeness and Reductions, Cook's theorem, Satisfiability
  • Approximation Algorithms
  • Gradient descent Algorithm (ML)
